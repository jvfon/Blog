use percentages instead of a fixed width.

By default a webpage is responsive, but when we add CSS we remove that feature from a webpage.

By default divs have a width of 100% of the parent. When we set fixed width in pixels or any other measurement, we remove the ability of the webpage to be responsive. Let's say the width of a div is set at 900px. Everything is fine when the browser is fully extended but when the browser is smaller than the 900px, we get side scrolling.

In contrast, if we set the width using percentages, the div will always be at a percentage of the parent no matter if the browser is big or small.
