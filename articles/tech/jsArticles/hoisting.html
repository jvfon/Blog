Hosting

Allow us to access functions and variables before they have been created. Function declarations and variable declarations are hoisted.

Function declarations

      
         sayHi();
   
         function sayHi() {
            console.log('Hey!');
         }

Result on the console: Hey!

It works because JS will move all the function declarations to the top of the file so they are available for use.

Variable hoisting

      
         console.log(age);
         age = 10

First JS creates the variable and then, it updates it.
