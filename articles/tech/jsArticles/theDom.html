The DOM

All the elements on a webpage.

The Window is the global scope in the browser.

Useful properties

On the console: ```window.location``` and ```location``` are the same thing because it's a property on the Window.

The ```location``` property gives us info about: host, host name, origin, etc.

```innerWidth``` tells us how wide the browser is.

The Window is the current open window including the browser bar, tabs, scroll bars, everything that is part of the browser window.
