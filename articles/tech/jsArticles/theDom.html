Useful properties

On the console: ```window.location``` and ```location``` are the same thing because it's a property on the Window.

The ```location``` property gives us info about: host, host name, origin, etc.

```innerWidth``` tells us how wide the browser is.

The Window is the current open window including the browser bar, tabs, scroll bars, everything that is part of the browser window.

DOM - Selecting elements

Putting <script src="name.js"></script> just before the </body> tag, ensures that the whole page loads first.

querySelector(); will only give you the first matching element. querySelectionAll(); will select all the matching elements. 
   var imgs = document.querySelectorAll(".item img");

Selecting an image inside a div with the class of "item".

We can also use querySelector on any other item. 

   var imgs = item2.querySelectorAll("img");
Selecting an image inside and element called "item2". Other selectors: ```js document.getElementById document.getElementByTagName ``` ## DOM - Elments Properties and Methods
