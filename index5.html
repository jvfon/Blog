Build a Modern Landing Page Series

Building a Landing page background

First start an HTML page like this: 

                     
         <!DOCTYPE html> 
         <html lang="en">  
            <head> 
               <meta charset="UTF-8"> 
               <meta name="viewport" content="width=device-width, initial-scale=1.0"> 
               <meta http-equiv="X-UA-Compatible" content="ie=edge"> 
               <title>Document</title>
            </head> 
            <body> 
                        
            </body> 
         </html>

index.html First inside the <body> element we are going to create a <header> element. 

                     
         <body>
            <header class="header">
               some text...
            </header>
         </body>

For now that’s all we need. We are now going to go to our style.css file.

Reset with this code: 

                     
         * { 
            padding: 0;
            margin: 0;
            box-sizing: border-box;
         }

The point of a reset is to eliminate variations due to browser defaults

Box-sizing: border-box; changes the box model so that the border and padding are no longer added to the total width and height that is specified for a box.

Font properties for the entire web page. The properties related to font are usually inherited 

                     
         Body {
            font-family: “Lato”, sans-serif;
            font-weight: 400;
            font-size: 16px;
            line-height: 1.7;
            color: #777;
         }

The font-family is the parent element of all the other elements and its properties will be inherited by all the children elements.

The font “Lato” is already linked from Google.com to the index.html file.

Line-height: 1.7; means 1.7 times the predefined line height.

The end result:

* { padding: 0; margin: 0; box-sizing: border-box; } body { font-family: "Lato", sans-serif; font-weight: 400; font-size: 16px; line-height: 1.7; color: #777; }

Now, we are going to edit the <header> element.

                     
         .header {
            height: 95vh;
            background-image: url(../img/hero.jpg);
            background-size: cover:
            background-position: top;
         }

height: 95vh makes the height of the <header> element 95% of the height of the viewport height.

background-image: url(../img/hero.jpg); two dots (..) to go up to the image folder and then write the file name.

background-size: cover; “cover” tries to fit the size of the image to the size of the box.

background-position: top; The top of the background stays in a fixed position when the web page is being resized. We can also use “center” and “bottom” for a different result.

To have a gradient all over the image, we use the background-image property. We are going to use the background-image property that we already have and add onto it. 

                     
         background-image: url(../img/hero.jpg);

Since we want the gradient to be on top. The gradient has to be specified first.

                     
         background-image: linear-gradient(to right bottom, #7ed56f, #28b485), url(../img/hero.jpg);

#7ed56f light green goes first.

#28b485 dark green goes last.

“to right bottom”, the gradient goes from the top to the right bottom corner.

Next we change from hex to rgba colors. On VS Code put the cursor on the hex number. A color picker window will show up, slide down the slider for the opacity until the number is at around 0.8.

                     
         background-image: linear-gradient(to right bottom, 
            rgba(127, 213, 112, 0.801), 
            rgba(39, 179, 133, 0.801)), 
            url(../img/hero.jpg);

The final code:

                     
            .header {
               height: 95vh;
               background-image: linear-gradient(to right bottom, 
                  rgba(127, 213, 112, 0.801), 
                  rgba(39, 179, 133, 0.801)), 
                  url(../img/hero.jpg);  
               background-size: cover:
               background-position: top;
            }

We are going to add padding around the entire body.

                     
         padding: 30px;

padding gets inherited.

                     
         .header {
            height: 95vh;
            background-image: url(../img/hero.jpg);
            background-size: cover:
            background-position: top;
            padding: 30px; 
         }
